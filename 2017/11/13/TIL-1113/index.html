Angular CLI 커맨드 정리

Angular CLI 설치

  • npm install -g angular-cli

프로젝트 생성

  • ng new 프로젝트명

포트변경

ng serve -port 4001

구성요소 추가

컴포넌트 추가

  • ng g c 컴포넌트명

디렉티브 추가

  • ng g d 디렉티브명

파이프 추가

  • ng g p 파이프명

서비스 추가

*ng g s 서비스명

빌드 및 실행

개발 서버 실행

  • ng serve

개발 서버 실행(watch)

  • ng serve -o

빌드

  • ng build
